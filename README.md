# Francisco Bachiller - Portfolio

Un portafolio interactivo con la estética completa de Windows XP. Navega a través de un sistema operativo funcional para conocer mis proyectos, habilidades y experiencia.

## Características

- **Interfaz Windows XP auténtica**: Escritorio, ventanas, taskbar, menú inicio y todos los elementos visuales clásicos
- **Sistema de ventanas funcional**: Drag & drop, minimizar, maximizar, cerrar
- **Navegación intuitiva**: Iconos de escritorio y menú inicio para acceder a diferentes secciones
- **Responsive**: Adaptado para diferentes tamaños de pantalla
- **Detalles nostálgicos**: Reloj en tiempo real, efectos hover, animaciones suaves

## Secciones

- **About Me**: Información sobre mí, mi filosofía y enfoque como desarrollador
- **Projects**: Mis proyectos destacados con detalles técnicos
- **Skills**: Tecnologías y habilidades que domino
- **Contact**: Información de contacto

## Tecnologías

- HTML5
- CSS3 (Vanilla CSS con gradientes y efectos Windows XP)
- JavaScript (ES6+)
- SVG para iconos personalizados

## Cómo usar

Simplemente abre `index.html` en tu navegador. No requiere instalación ni dependencias.

## Características técnicas

- Sistema de gestión de ventanas con z-index dinámico
- Drag & drop nativo sin librerías
- Reloj en tiempo real
- Gestión de estado de ventanas (minimizado, maximizado, activo)
- Taskbar funcional con items dinámicos
- Menú inicio interactivo
